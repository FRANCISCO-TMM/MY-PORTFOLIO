Academic Planner

The dashboard I actually use.

Add a task, move it across the board, and it saves right in your browser — no login required.

Why I built this

Sticky notes and memory weren't cutting it once I had coursework, CTF practice, and a capstone project running at the same time. This planner is my own lightweight Kanban board — built with vanilla HTML, CSS, and JavaScript as part of my COS 106 Lab Assessment, and something I genuinely use every week.

How it's built

No backend, no database — tasks are saved to your browser's localStorage, so they persist between visits on the same device. Everything is DOM manipulation and array logic: add, move, complete, and delete a task, and the board and stats re-render instantly.
